Intervalidus's main traits and classes for defining immutable dimensional data, including versioned and multivalued data:

  • Base traits for all immutable dimensional data.
  • Classes defining immutable data in one, two, and three dimensions.

Data that may have multiple values valid in different intervals, conceptually similar to a multimap. These intervals may represent when data are valid in time or over certain versions ranges or whatever. When queried, values are returned as a set.
