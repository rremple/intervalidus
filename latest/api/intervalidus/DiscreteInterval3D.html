case class DiscreteInterval3D[T1, T2, T3](horizontal: DiscreteInterval1D[T1], vertical: DiscreteInterval1D[T2], depth: DiscreteInterval1D[T3])(implicit evidence$1: DiscreteValue[T1], evidence$2: DiscreteValue[T2], evidence$3: DiscreteValue[T3]) extends IntervalLike[DiscreteDomain3D[T1, T2, T3], DiscreteInterval3D[T1, T2, T3]]

A two-dimensional interval over a contiguous set of discrete values in T1 and T2. See https://en.wikipedia.org/wiki/Interval_(mathematics) for more information.

Type parameters

T1

a discrete value type for this interval's horizontal domain

T2

a discrete value type for this interval's vertical domain

Value parameters

depth

the third dimension interval: the (right-handed) depth Z-axis (behind to in front)

horizontal

the abscissa interval: the horizontal X-axis (from left to right)

vertical

the ordinate interval: the vertical Y-axis (from lower to upper)
