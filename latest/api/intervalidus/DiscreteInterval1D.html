DiscreteInterval1D

See theDiscreteInterval1D companion object
case class DiscreteInterval1D[T](start: DiscreteDomain1D[T], end: DiscreteDomain1D[T])(implicit evidence$1: DiscreteValue[T]) extends DiscreteIntervalLike[DiscreteDomain1D[T], DiscreteInterval1D[T]]

A one-dimensional interval over a contiguous set of discrete values in T. See https://en.wikipedia.org/wiki/Interval_(mathematics) for more information.

Type parameters

T

a discrete value type for this interval's domain (e.g., Int, LocalDate) -- boundaries of the interval are defined in terms of DiscreteDomain1D[T] given the type class DiscreteValue[T].

Value parameters

end

the "supremum", i.e., the right (or above or front, depending on context) boundary of the interval (inclusive) -- must be greater than or equal to the start

start

the "infimum", i.e., the left (or below or back, depending on context) boundary of the interval (inclusive)

infix override def excluding(that: DiscreteInterval1D[T]): ExclusionRemainder

Excludes that interval from this one. There are three possible outcomes:

Excludes that interval from this one. There are three possible outcomes:

  1. this interval is a subset of that one, so once it is excluded, nothing remains. Remainder.None is returned.
  2. that either lies outside of this or has a simple edge intersection (only contains the start or the end, but not both), and a single interval remains. Remainder.Single is returned.
  3. that interval is a proper subset of this one, containing neither the start nor the end of this, so this interval gets split, and two intervals remain. Remainder.Split is returned

Value parameters

that

the interval to exclude.
