DataLike

intervalidus.DimensionalBase.DataLike
trait DataLike[V, D <: DomainLike[D], I <: IntervalLike[D, I], Self <: DataLike[V, D, I, Self]] extends PartialFunction[D, V]

A value that is valid in some discrete interval. This defines a partial function where all domain elements that are part of the interval map to the specified value.

Type parameters

D

the type of discrete domain used in the discrete interval assigned to each value (the domain).

I

the type of discrete interval in which the value is valid.

Self

F-bounded self type.

V

the type of the value managed as data (the codomain).
