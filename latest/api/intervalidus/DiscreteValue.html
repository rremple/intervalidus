DiscreteValue

intervalidus.DiscreteValue
See theDiscreteValue companion object
trait DiscreteValue[T] extends Ordering[T]

Template for a type class with operations on a discrete value. A discrete value is

  1. finite, with a max and min value (think Int with its MaxValue and MinValue)
  2. totally ordered (this type class extends the Ordering type class, requiring a compare method).
  3. have predecessor and successor functions, which are properly defined on x: max < x < min.

Having predecessors and successors defined this way avoids issues where the natural successor functions of the underlying types behave unexpectedly/inconsistently on the boundaries, e.g., how Int.MaxValue + 1 and Int.MinValue - 1 silently wrap around to each other, vs. how both LocalDate.MAX.plusDays(1) and LocalDate.MIN.minusDays(1) throw a DateTimeException.
