object BigIntegerDiscreteValue extends DiscreteValue[BigInteger]

Type class for big integers as discrete values.

Type class for big integers as discrete values.

Attributes

Note

per https://docs.oracle.com/javase/8/docs/api/java/math/BigInteger.html "BigInteger constructors and operations throw ArithmeticException when the result is out of the supported range of -2Integer.MAX_VALUE (exclusive) to +2Integer.MAX_VALUE (exclusive)." However, on the practical side of things, this number is so large that just rendering it as a string can easily cause OOM in the string builder! So maybe override this and pick your own "biggest" BigInteger based on how you plan to use it rather than this theoretically biggest one...
