case class ValidData2D[V, R1, R2](value: V, interval: DiscreteInterval2D[R1, R2])(implicit evidence$1: DiscreteValue[R1], evidence$2: DiscreteValue[R2]) extends ValidDataLike[V, DiscreteDomain2D[R1, R2], DiscreteInterval2D[R1, R2], ValidData2D[V, R1, R2]]

A value that is valid in a two-dimensional discrete interval. Conceptually, this defines a partial function where all domain elements that are part of the 2D interval map to the value.

Type parameters

R1

the type of the horizontal discrete interval.

R2

the type of the vertical discrete interval.

V

the type of the value managed as data (the codomain).

Value parameters

interval

the discrete interval in which the value is valid.

value

value that is valid in this 2D interval.
