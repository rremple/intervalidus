type DimensionIndex = Int & Singleton

A numeric reference to a specific dimension. Dimensions are indexed starting at 0. For example, a three-dimensional domain would have dimensions indexed by 0, 1, and 2.

A numeric reference to a specific dimension. Dimensions are indexed starting at 0. For example, a three-dimensional domain would have dimensions indexed by 0, 1, and 2.

Attributes

Note

Dimension indexes are integer singletons, i.e. literal values that must be known at compile time.
